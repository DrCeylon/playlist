## Before opening an issue

1. Search [existing issues](https://github.com/DrCeylon/playlist/issues)
2. State your environment: OS, Python version, Resonance commit or tag
3. For import problems: macOS version, Music.app version, sync library enabled
4. Attach relevant logs from `reports/` (redact personal data)

## What we support

| Component | Support level |

## Response expectations

This is a community open-source project maintained in spare time. There is no SLA. Critical security reports are prioritized per [SECURITY.md](SECURITY.md).

## Commercial support

Not available. For enterprise inquiries, open a discussion issue describing your needs.
